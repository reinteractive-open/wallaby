# Support:
- All 37 postgres data types that ActiveRecord supports: (this can be found from `ActiveRecord::ConnectionAdapters::PostgreSQLAdapter::NATIVE_DATABASE_TYPES.keys`)

    - binary
    - bit
    - bit_varying
    - boolean
    - box
    - cidr
    - circle
    - citext
    - date
    - daterange
    - datetime
    - decimal
    - float
    - hstore
    - inet
    - int4range
    - int8range
    - integer
    - json
    - jsonb
    - line
    - lseg
    - ltree
    - macaddr
    - money
    - numrange
    - path
    - point
    - polygon
    - string
    - text
    - time
    - tsrange
    - tstzrange
    - tsvector
    - uuid
    - xml

- Additional types:

    - color
    - email
    - sti
    - password
    - raw (index/show)
    - dropdown (form)
    - markdown (form)
    - file (form)
